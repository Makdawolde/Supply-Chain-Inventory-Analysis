# Supply-Chain-Inventory-Analysis
This Power BI project presents a comprehensive suite of interactive dashboards designed to provide deep insights into supply chain performance across three key dimensions: Overview, Product Insight, and Supplier Insight.

Key Features
Fully interactive with filters for SKU and Supplier Name.
Enables granular analysis for strategic decision-making in procurement, logistics, and inventory management.
Empowers stakeholders to optimize cost, quality, and delivery performance across the supply chain.
